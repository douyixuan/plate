# Changelog

## [0.0.1] - 2025-03-15

### Added
- Initial release of Plate Markdown Editor
- Basic markdown editing support
- Custom editor for .md files
- Integration with VSCode markdown files

### Features
- WYSIWYG markdown editing
- Headings (H1, H2, H3)
- Paragraph support
- Blockquote support

### Planned Features

- Lists
